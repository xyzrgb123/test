# display grid (CSS)
---

## Терминалогия
- **контейнер** = *контейнер в котором и создается сетка*
- **элемент** = *прямой потомок грид-контейнера*
- **линии** = *направляющие для привязки элементов*
- **ячейка** = *пространство между соседствующими линиями*
- **полоса** = *пространство между двумя соседними и параллельными линиями
- **область** = *пространство ограниченное двумя вертикальными и двумя горизонтальными линиями*

## Контейнер
---
```
display grid
```
- ширина = вся доступная ширина
- высота = по собственному содержимому

### Набор размеров для строк или столбцов
---
задает модель колонок,  
задай через пробел размеры в тех единицах которые надо.  
```
200px auto 10wv
```
перед каждым размером можно (не обязательно) в квадратных скобках именовать линии.  
```
[john] 200px [den] auto [silvia] 10wv
```
повторяющиеся наборы размеров можно объединить
```
repeat(3, 300px) repeat(2, 100px)
```
1fr - единица измерения... сколько частей от еще не использованного размера.  
в примере, первый размер - 2 части от целого, а второй - одна
```
2fr 1fr
```
**grid-template-columns** - *задает набор ширин колонок*  
**grid-template-rows** - *задает набор высот строк*  

### grid-template-areas
---
задает название областей сетки через пробел
```
menu content footer
```

### Полосы
---
row-gap - ширина горизонтальных полос
colomn-gap - ширина вертикальных полос
gap - ширина полос

### justify-content
---
выравнивание всей сетки по горизонтале  
start - к левому краю  
end - к правому краю  
center - по-центру  
stretch - растянуть на всю ширину (по умолчанию)  
space-beetween - равномерное распределение с минимальными ширинами элементов  
space-around - похож на предидущий, но с добалением пространств по краям равным половине ширин между элементами  
space-evently = как и around но в начале и конце такие же пространства как и между элементами  

### align-items
---
выравнивание по вертикале
normal - по высотам содержимого  
start - к верху  
end - к низу  
center - к центру  
stretch - тоже что normal

### place-items
---
объединанеие свойств align-items и justify-content  
задай оба этих свойства через пробел (сперва алигн)...или одно значение если совпадуха
